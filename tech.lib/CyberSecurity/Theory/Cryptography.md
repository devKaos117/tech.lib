## Definition

## Encryption
Two-way function, in which data is obfuscated or deobfuscated with at least one key. Encrypted data is known as a [_ciphertext_]

### Symmetric
_Symmetric encryption_ algorithms use the same key for both encryption and decryption

### Asymmetric
_Asymmetric encryption_ uses distinct key pairs containing private and public keys

## Hashing
Hashing algorithms are a one-way functions that produces a hash (or _digest_), a practically unique fixed-length hexadecimal value that represents the original _plaintext_ (the variable-sized input data)
Extremely rare _hash collision_ in which two input values result in the same hash value
