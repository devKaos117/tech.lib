## Definition
Microsoft Azure is a cloud computing platform focused on corporative services

### Azure Arc
set of technologies that helps manage your cloud environment whether it's a public cloud solely on Azure, a private cloud in your datacenter, a hybrid configuration, or even a multi-cloud environment running on multiple cloud providers at once

### VMware
Azure VMware Solution lets you run your VMware workloads in Azure with seamless integration and scalability
