> [!info] Print the full filename of the current working directory

## Command
```txt
pwd [-P|-L]

-L, --logical
	Use PWD from environment, even if it contains symlinks 
-P, --physical
	Avoid all symlinks 

--help
	Display the help information and exit 
--version
	Output version information and exit
```

## Exit Status
- 0 - success
- n - failure
