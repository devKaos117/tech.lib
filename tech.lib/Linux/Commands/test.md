## Command
```txt
test [OPTION] [EXPRESSION]

--help
	Display the help information and exit 
--version
	Output version information and exit

EXPRESSIONS
```

## Exit Status
- 0 - Expression returned true
- 1 - Expression returned false
- 2 - Error
