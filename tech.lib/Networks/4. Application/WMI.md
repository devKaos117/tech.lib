## Definition
Microsoft's implementation and also an extension of the CIM (Common Information Model), core functionality of the standardized WBEM (Web-Based Enterprise Management)
allows for read and write access to almost all settings on Windows systems
typically accessed via PowerShell, VBScript, or the WMIC (Windows Management Instrumentation Console)
TCP port 135
