## Definition
runs on [[TCP]] ports 20 and 21 for data transfer and connection management
runs on clear text
has active and passive modes
the passive mode was built to work with restrictive firewalls on the client side
[Commands](https://en.wikipedia.org/wiki/List_of_FTP_commands)
[Server Response Codes](https://en.wikipedia.org/wiki/List_of_FTP_server_return_codes)

[TFTP]{Trivial File Transfer Protocol} is a simpler version that does not support [[IAM#Authentication|AuthN]] and works over [[UDP]] nor directory listing

[SFTP]{SSH File Transfer Protocol} and [FTPS]{File Transfer Protocol Secure}
