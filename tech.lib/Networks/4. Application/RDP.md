## Definition
developed by Microsoft for remote access to a computer running the Windows operating system
Working at the application layer, it allows for display and control commands to be transmitted via the GUI encrypted over IP networks
Supports SSL/TLS but allows for RDP Security
TCP/UDP port 3389
Must have firewall permissions and NAT (Network address translation) properly setted up
